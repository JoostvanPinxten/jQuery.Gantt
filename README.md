[Demo and Documentation](http://joostvanpinxten.github.com/jQuery.Gantt/)
==============

This variant of the jQuery Gantt Chart is a simple chart that implements gantt functionality as a jQuery component. It does not use dates, but plain data.

It's able to:

 - Read json data
 - Paging/scrolling results
 - Display different colours for each task
 - Display short description as hints
